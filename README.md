# LeetPal

LeetPal is a Chrome extension for people who practice coding problems on LeetCode and want their practice to feel less random.

Most learners do not struggle because there are too few problems. They struggle because it is hard to know what to solve next. LeetCode tags are broad, lists are static, and "similar problems" often share a topic without training the same underlying idea.

LeetPal helps with that exact moment.

## What LeetPal Does

When you open a LeetCode problem, LeetPal shows a small companion panel with a few carefully chosen similar problems.

Each recommendation is meant to answer:

> What should I solve next if I want to reinforce the same kind of thinking?

Instead of sending users into a huge topic list, LeetPal keeps the experience focused. It gives a short set of nearby practice problems so the next solve has a clear purpose.

## Why It Is Useful

Interview prep is not just about increasing your solved count. The real skill is recognizing the same pattern when it appears in a different form.

LeetPal is built around that idea:

- Practice the same underlying technique across different problems.
- Avoid jumping randomly between broad tags.
- Build pattern recognition through deliberate repetition.
- Spend less time deciding what to solve next.
- Keep working inside LeetCode without opening another tool.

The goal is simple: turn one problem into a focused practice path.

## How The Extension Feels

LeetPal stays lightweight. It appears as a small floating button on LeetCode problem pages and opens into a compact panel when needed.

Inside the panel, users can see:

- The current problem they are working on.
- The closest recommended practice matches.
- A match indicator for each recommendation.
- Difficulty and problem number.
- An optional pattern note when the user wants more context.

This keeps the extension useful without taking over the page or turning into a noisy assistant.

## The Main User Flow

1. Open a LeetCode problem.
2. Open LeetPal from the floating button.
3. Review the top similar problems.
4. Pick one recommendation as the next practice problem.

If a problem is not available in LeetPal yet, the extension can quietly queue it for review so coverage improves over time.

## What Makes LeetPal Different

LeetPal is not trying to be a full AI tutor, code explainer, or mock interview platform.

Its main job is narrower:

> Help users choose better next problems.

That focus matters. Hints can help you finish one problem, but better sequencing helps you learn the pattern behind it. LeetPal is designed for users who want practice to compound instead of becoming another grind.

## Best Fit Users

LeetPal is especially useful for:

- Students preparing for coding interviews.
- People who have solved some LeetCode problems but still feel inconsistent.
- Users who understand solutions but struggle to recognize patterns later.
- People following problem lists who want sharper follow-up recommendations.
- Learners who want guidance without spoiler-heavy AI help.

A good LeetPal user is someone who has thought:

> I solved this, but I am not sure I learned it.

LeetPal helps answer what to do next.

## Product Promise

LeetPal helps LeetCode practice become more intentional.

It recommends problems that feel meaningfully related, keeps track of what the user has already solved, and gives just enough context to guide the next practice decision without distracting from the solve itself.

In one line:

> LeetPal helps you stop guessing what to solve next and start building real pattern recognition.
